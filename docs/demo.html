SlideGear → Demo One

This page shows three slider devices with different settings.

Try your browser with CTRL-U to inspect the details. The three settings are like listed below.

  • Slider one has no image source specified, so the slider shows the built-in fallback images. (This feature is to be dropped)
  • Slider two has image sources defined inside the page.
  • Slider three has image sources specified in JSON file demo.html.json.
